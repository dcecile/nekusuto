Connecting Game Devs and Let’s Players

Nekusuto matches great games with great Let’s Players. You get all of the contact and background info you need to start the conversation.

How does it work for game devs?

Select your game’s genres.

Nekusuto finds Let’s Players that enjoy playing similar games.

You get Twitter handles and recent videos for the top-ranked matches.

How does it work for Let’s Players?

Select your favourite genres.

Nekusuto finds game devs actively working on similar games.

You get Twitter handles and game details for the top-ranked matches.

How much does it cost?

Free for game devs with less than 500 Steam reviews.

Free for Let’s Players with less than 5000 Youtube subscribers.

Others: let’s chat and see what’s fair.
